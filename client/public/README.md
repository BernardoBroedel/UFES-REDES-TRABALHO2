# Pasta de Assets Estáticos

Esta pasta contém arquivos estáticos que serão servidos diretamente pelo Vite.

## Música de Fundo

Para adicionar a música de fundo:

1. Coloque o arquivo de música (formato MP3) nesta pasta com o nome: `background-music.mp3`

2. O arquivo será automaticamente carregado e reproduzido em loop quando a aplicação iniciar.

### Nome do arquivo esperado:
- `background-music.mp3`

### Formatos suportados:
- MP3 (recomendado)
- OGG
- WAV

### Nota:
O volume da música está configurado para 10% do volume máximo. Você pode ajustar isso no arquivo `App.tsx` alterando o valor de `audio.volume`.
